## 🖼️ CIFAR-10 Image Classification using CNN

This is a beginner-friendly deep learning project where I implemented a Convolutional Neural Network (CNN) to classify images from the CIFAR-10 dataset.

---

## 📌 About

This project is part of my learning journey in Deep Learning.  
I built a CNN model to classify images into multiple object categories.

---

## 📊 Dataset

- CIFAR-10 dataset  
- Contains 10 different classes  
- Includes objects like airplanes, cars, birds, cats, etc.
